Hello World

Published 2026-03-15

Welcome to PureSimpleHTTPServer — the static file server that gets out of your way. If you have ever struggled to remember the right Python one-liner, wrestled with Node version managers, or just wanted something that works immediately with no setup, this is the tool for you.

Step 1 — Download the binary

Head to the Releases page and grab the archive for your operating system. Unzip it anywhere you like. There is nothing else to install. 

# macOS / Linux
curl -L https://example.com/releases/v1.5.0/pshs-macos-arm64.tar.gz | tar xz
chmod +x PureSimpleHTTPServer

On Windows, simply unzip the .zip and double-click PureSimpleHTTPServer.exe, or run it from PowerShell.

Step 2 — Organise your files

PureSimpleHTTPServer serves files from a root directory — by default a folder called wwwroot next to the binary. Place your HTML, CSS, JavaScript, and images inside that folder: 

my-project/
  PureSimpleHTTPServer      ← the binary
  wwwroot/
    index.html
    about.html
    assets/
      style.css
      app.js
      logo.png

You can also pass any directory with --root if you prefer a different layout.

Step 3 — Start the server

Open a terminal, navigate to your project folder, and run:

./PureSimpleHTTPServer --port 8080 --root ./wwwroot

You should see a startup message like: 

PureSimpleHTTPServer v1.5.0 listening on http://0.0.0.0:8080
Serving: /Users/yourname/my-project/wwwroot

Step 4 — Open your browser

Navigate to http://localhost:8080. Your index.html will be served automatically. Any file inside wwwroot is accessible at its relative path — for example, wwwroot/about.html is available at /about.html.

If you enable the --clean-urls flag, the server will also serve about.html when the browser requests /about — no rewrite.conf required for simple cases.

What's next?

Once you are comfortable with the basics, check out the URL Rewriting guide to learn how to set up clean, SEO-friendly URLs, permanent redirects, and regex-based routing. For a full list of CLI flags and configuration options, see the Documentation.

Happy serving!
