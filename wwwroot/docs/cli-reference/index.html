CLI Reference

All command-line flags for PureSimpleHTTPServer v1.5.0. Flags may appear in any order and can be freely combined.

./PureSimpleHTTPServer [flags...]

All flags

FlagTypeDefaultDescription
--port NInteger8080TCP port to listen on
--root DIRPathwwwroot/Document root directory
--browseFlagoffEnable directory listing when no index file
--spaFlagoffServe root index.html for all 404s
--log FILEPathdisabledAccess log path (Apache Combined Log Format)
--error-log FILEPathdisabledError log path
--log-level LEVELEnumwarnError log verbosity: none / error / warn / info
--log-size MBInteger100Rotate logs when they reach this size in MB (0 = disabled)
--log-keep NInteger30Max archived log files to keep
--no-log-dailyFlagoffDisable automatic midnight log rotation
--pid-file FILEPathdisabledWrite PID to file at startup; deleted on clean exit
--clean-urlsFlagoffServe /page.html when /page is requested
--rewrite FILEPathdisabledLoad URL rewrite/redirect rules from file

Examples

Minimal — serve current directory on port 3000

./PureSimpleHTTPServer --root . --port 3000

SPA (React / Vue / Angular)

./PureSimpleHTTPServer --root ./dist --spa --port 3000

Static site with clean URLs and access logging

./PureSimpleHTTPServer \
  --root ./public \
  --clean-urls \
  --log /var/log/pshs/access.log \
  --error-log /var/log/pshs/error.log

Production with log rotation and PID file

./PureSimpleHTTPServer \
  --root /srv/www \
  --port 8080 \
  --log /var/log/pshs/access.log \
  --error-log /var/log/pshs/error.log \
  --log-level warn \
  --log-size 100 \
  --log-keep 30 \
  --pid-file /run/pshs.pid \
  --rewrite /etc/pshs/rewrite.conf

Legacy positional argument

A bare port number without a flag name is accepted for backwards compatibility:

./PureSimpleHTTPServer 9000   # same as --port 9000

Prefer --port in scripts and service files.
