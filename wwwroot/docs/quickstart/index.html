Quick Start

Get PureSimpleHTTPServer running in under two minutes.

1. Download or build

Download the pre-built binary for your platform, or compile from source:

pbcompiler -cl -t -o PureSimpleHTTPServer src/main.pb

2. Arrange your files

Put the binary next to your site files in a wwwroot/ folder:

my-site/
  PureSimpleHTTPServer   ← binary
  wwwroot/
    index.html
    style.css
    ...

On macOS/Linux, make it executable:

chmod +x PureSimpleHTTPServer

3. Start the server

./PureSimpleHTTPServer

Open http://127.0.0.1:8080 in your browser. That's it.

Common options

FlagEffectExample
--port NUse a different port--port 3000
--root DIRServe from any directory--root ./dist
--browseEnable directory listing--browse
--spaSPA mode (serve index.html for 404s)--spa
--clean-urlsServe /page.html at /page--clean-urls
--log FILEWrite access log--log access.log
